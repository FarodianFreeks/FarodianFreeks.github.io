# Farodian Freeks | Virtual CV

A responsive personal portfolio website created to present my academic background, technical skills, software projects, current learning interests, professional links, and downloadable CV.

## Live Website

**Virtual CV:** https://farodianfreeks.github.io/

## About the Project

This website serves as my online professional portfolio and Virtual CV. It was designed to provide recruiters, lecturers, and other professionals with a clear overview of who I am, what I am studying, the technologies I work with, and the projects I have built.

The website complements my professional CV, GitHub profile, and LinkedIn profile by bringing all of my work together in one accessible location.

## Main Features

- Responsive layout for desktop, tablet, and mobile devices
- Mobile navigation menu
- Light and dark colour themes
- Smooth scrolling and reveal animations
- Interactive buttons and visual effects
- Downloadable professional CV
- Direct links to GitHub and LinkedIn
- Project descriptions with links to their repositories
- Accessible navigation labels and keyboard-friendly controls
- Reduced-motion support for users who prefer fewer animations
- Browser-friendly fallbacks for visual effects

## Running the Website Locally

### Option 1: Open the HTML file

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.

### Option 2: Use Visual Studio Code and Live Server

1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

## Cloning the Repository

```bash
git clone https://github.com/FarodianFreeks/FarodianFreeks.github.io.git
cd FarodianFreeks.github.io
```

Then open `index.html` in a browser or launch it through Live Server.

## Responsive Design

The website uses responsive layouts and media queries so that its content remains readable and easy to navigate across:

- Desktop computers
- Laptops
- Tablets
- Large mobile phones
- Small mobile phones

The mobile version includes a dedicated navigation menu, adjusted typography, touch-friendly controls, and layouts that prevent horizontal overflow.

## Browser Compatibility

The website is designed for current versions of:

- Google Chrome
- Safari
- Microsoft Edge
- Mozilla Firefox

The project uses standard HTML, CSS, and JavaScript. Fallback styling is included for browsers that do not support certain glass or blur effects.

## Accessibility Considerations

The website includes:

- Semantic HTML sections
- Descriptive link and button labels
- Keyboard-accessible navigation
- Visible focus states
- Reduced-motion support
- Readable colour contrast
- Mobile-friendly touch targets
- Alternative text for meaningful images
